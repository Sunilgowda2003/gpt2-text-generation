
# 🤖 GPT-2 Text Generation Project

## 📌 Description
This project demonstrates text generation using the GPT-2 model.
The model is fine-tuned on a custom dataset and generates meaningful text based on input prompts.

---
 
## 🚀 Features  
- Uses pre-trained GPT-2 model 
- Fine-tuning on custom dataset 
- Generates human-like text  
- Simple and beginner-friendly implementation 

---

## 🛠️ Technologies Used
- Python 
- Hugging Face Transformers
- PyTorch
- Datasets Library
